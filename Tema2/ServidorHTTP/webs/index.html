Proyecto Java (NetBeans): Servidor HTTP Multihilo

Este proyecto ha sido desarrollado en Java utilizando el entorno de desarrollo NetBeans.

Su principal objetivo es la implementación de un servidor HTTP básico con soporte para múltiples conexiones simultáneas mediante programación multihilo.

Se han utilizado clases como ServerSocket y Thread para gestionar las peticiones entrantes de forma concurrente.

Este índice resume el propósito y tecnología del proyecto.
